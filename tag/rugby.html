Building a web scraper for international rugby scoring

Published: Thu 04 January 2018
By John Boudreaux

In General.

tags: Rugby Sports Web Scraper

ESPN has a database of all international rugby scores for matches in the present day all the way back through the 1950s and even some earlier matches. Using Jupyter notebooks, I wanted to go through the basic steps of building a web scraper to get this data into a more friendly format (.csv) so that I can do more analysis with it in Python or R. This web scraper is built using Python and BeautifulSoup. Let's start by taking a look at the URL of a relevant query of the last 10 years worth of international data.

URL: http://stats.espnscrum.com/statsguru/rugby/stats/index.html?class=1;page=;spanmax1=08+Nov+2017;spanmax2=08+Nov+2017;spanmin1=08+Nov+2007;spanmin2=08+Nov+2015;spanval1=span;spanval2=span;template=results;type=player;view=match

We can define empty arrays that we'll store all our values in for later. Also, we should go ahead and import relevant libraries to start.

With that out of the way, now we can get to the actual scraper. Let's define a string, that we can iterate through for each individual page for all the pages. There were 1601 pages the last time I checked.

In [2]:
def table_url_page(page):
	return('http://stats.espnscrum.com/statsguru/rugby/stats/index.html?class=1;page=' + str(page) +
	';spanmax1=08+Nov+2017;spanmax2=08+Nov+2017;spanmin1=08+Nov+2007;spanmin2=08+Nov+2015;' + 
	'spanval1=span;spanval2=span;template=results;type=player;view=match')

For the sake of this being an example, I'll only do one page. But to create this into a web scraper to get information from all 1061 pages, you would simply need to iterate over all the page numbers and store the values in the arrays we defined in cell 1. It would also be good practice to have a slight delay between each iteration, to not create too many requests at one time.

But let's go ahead and start with the first page, page 1.

We now have the framework of what we need to do in order to create a csv. All we need to do is loop for all the table entries on the page, within a larger loop of going through all the pages. We can store each entry as an item in an array, and then create a data frame using pandas. We can then call .to_csv and we'll have our data frame.
